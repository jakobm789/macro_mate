## Running Tests

1. Install the Flutter SDK version used for this project:

```bash
./scripts/setup_flutter.sh
```

2. Add the SDK to your PATH as indicated by the script output or run:

```bash
export PATH="$(pwd)/flutter_sdk/bin:$PATH"
```

3. Run the tests:

```bash
flutter test
```
